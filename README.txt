[PT-BR] INSTRUÇÕES DE INSTALAÇÃO:

Este é um port do jogo Annelids para o console portátil R36S baseado no 
ecossistema PortMaster. Por motivos de direitos autorais, os arquivos 
comerciais do jogo NÃO estão incluídos neste pacote. Você precisa fornecê-los.

REQUISITOS:
- Arquivo APK do jogo Annelids (Versão antiga estável recomendada: 1.116.6).

PASSO A PASSO:
1. Extraia o conteúdo deste arquivo `.zip` dentro da pasta 'roms/ports/' 
   no cartão SD do seu portátil.
2. No seu celular ou computador, mude a extensão do seu arquivo APK do 
   Annelids para `.zip` e descompacte-o.
3. Copie a pasta 'assets' inteira extraída do APK para dentro da pasta:
   'roms/ports/annelids/'
4. Acesse a pasta 'lib/arm64-v8a/' que foi extraída do APK, copie o arquivo 
   'libannelids.so' e cole-o também dentro da pasta:
   'roms/ports/annelids/'
5. Coloque o cartão de volta no R36S, navegue até a seção 'Ports' no menu 
   e inicie o jogo!

NOTA DE PERMISSÃO (Caso o jogo não abra):
Se o jogo fechar imediatamente, use a opção "Fix Permissions" dentro do menu 
do PortMaster no seu portátil para corrigir as permissões do sistema Linux.

[EN] INSTALLATION INSTRUCTIONS:

This is an Annelids game port for the R36S handheld console based on the 
PortMaster ecosystem. Due to copyright policies, commercial game files 
are NOT included in this package. You must provide them yourself.

REQUIREMENTS:
- Annelids game APK file (Recommended stable old version: 1.116.6).

STEP-BY-STEP:
1. Extract the contents of this `.zip` file into the 'roms/ports/' folder 
   on your handheld's SD card.
2. On your phone or PC, rename your Annelids APK file extension to `.zip` 
   and extract it.
3. Copy the entire 'assets' folder extracted from the APK into:
   'roms/ports/annelids/'
4. Go to the extracted 'lib/arm64-v8a/' folder from the APK, copy the 
   'libannelids.so' file and paste it into:
   'roms/ports/annelids/'
5. Put the SD card back into your R36S, navigate to the 'Ports' section 
   in the menu, and launch the game!

PERMISSION NOTE (If the game doesn't launch):
If the game closes instantly, use the "Fix Permissions" option inside the 
PortMaster menu on your handheld to fix Linux system execution rights.
